Front-end Web Development

Teaching people front-end web development

HTML5

HTML5 is the latest version of the language of the web: the Hyper-Text Markup Language.

It is used to define the structure of your website's content.

CSS3

CSS3 is the third version of the styling language of the web: the Cascading Style Sheets language.

CSS is used to style websites.

Bootstrap

Bootstrap is an open-source template designed to make front-end development easier.

Bootstrap can be integrated in your website to help you make a beautiful responsive website.
